CoNavic

CoNavic is a browser extension that brings the power of ChatGPT and browser automation to your fingertips. Instantly access AI assistance, manage tabs, and organize bookmarks using natural language, all from any page.

Key Features

  • Open the chat bar anywhere with Ctrl+Shift+K
  • Ask ChatGPT questions and get instant, markdown-formatted answers
  • Automate browser tasks: open/close tabs, group tabs, move tabs, and more
  • Organize bookmarks and folders using AI commands
  • All actions are performed securely in your browser

How to Use

  1. Press Ctrl+Shift+K to open the chat bar
  2. Type your question or command (e.g., "Open GitHub in a new tab")
  3. Press Enter to send
  4. See responses and actions performed instantly
